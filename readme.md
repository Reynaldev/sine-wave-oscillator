# Sine Wave Oscillator
A program that creates sine wave oscillator

# Compiling
```
git clone https://github.com/Reynaldev/sine-wave-oscillator.git
cd ./sine-wave-oscillator
make
```

# Usage
`./swo [duration]`

And there will be a WAV audio file
